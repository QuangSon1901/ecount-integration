Hướng dẫn cài đặt

1
Tải xuống Extension
Nhấn vào nút Tải xuống Extension ở trên để tải file .zip về máy tính.
2
Giải nén file
Giải nén file down-label-ecount-extension-tool.zip vào thư mục riêng (ví dụ: C:\ChromeExtensions\Tool)
3
Mở Chrome Extensions
Truy cập chrome://extensions/ trên trình duyệt Chrome
4
Bật Developer Mode
Bật công tắc Developer mode ở góc trên bên phải
5
Load Extension
Nhấn nút Load unpacked và chọn thư mục đã giải nén
6
Xác nhận cài đặt
Extension sẽ xuất hiện trong danh sách. Đảm bảo extension được bật.
Lưu ý quan trọng
  • Không xóa thư mục đã giải nén sau khi cài đặt
  • Extension cần quyền truy cập vào trang ECount để hoạt động
  • Đảm bảo đăng nhập ECount trước khi sử dụng extension
  • Nếu gặp lỗi, thử tắt và bật lại extension

Cách sử dụng

  1. Theo dõi trạng thái đơn hàng theo thời gian thực
  2. Cập nhật tracking number tự động từ hệ thống vận chuyển
  3. Xử lý hàng loạt nhiều đơn hàng cùng lúc
  4. Xuất báo cáo và thống kê đơn hàng
